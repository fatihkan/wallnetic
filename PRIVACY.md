# Wallnetic Privacy Policy

**Last updated:** March 2026

## Overview

Wallnetic is a macOS application that allows you to use video files as live desktop wallpapers. We are committed to protecting your privacy.

## Data Collection

**We do not collect any data.**

Wallnetic:
- Does not collect personal information
- Does not track usage or analytics
- Does not connect to external servers
- Does not require an account
- Works entirely offline

## Local Storage

All data is stored locally on your Mac:

| Data Type | Location |
|-----------|----------|
| Video files | `~/Library/Application Support/Wallnetic/Library` |
| Preferences | macOS UserDefaults |
| Collections | Local JSON file |

We do not have access to any of this data. There is no cloud sync or external storage.

## Third-Party Services

Wallnetic does not use any third-party services, analytics, or tracking tools.

## Permissions

Wallnetic may request the following permissions:

| Permission | Purpose |
|------------|---------|
| Screen Recording | Required to display wallpaper on desktop |
| File Access | To import and manage video files |

These permissions are used solely for the app's core functionality.

## Children's Privacy

Wallnetic does not collect any information from anyone, including children under 13.

## Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated revision date.

## Contact

If you have any questions about this Privacy Policy, please contact us:

- **Email:** support@wallnetic.app
- **GitHub:** [github.com/fatihkan/wallnetic](https://github.com/fatihkan/wallnetic)

## Your Rights

Since we don't collect any data, there is no personal data to access, modify, or delete. You can:

- Delete all local data by uninstalling the app
- Clear your wallpaper library through the app
- Remove app permissions in System Settings
